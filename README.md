# 📌 Interactive Digital Vision Board

A highly aesthetic, corkboard-style digital vision board built to showcase dynamic front-end interactions, fluid layout systems, and native browser optimizations. This tool allows users to map out goals, upload imagery, and seamlessly organize milestones across multiple lifestyle domains.

---

## ✨ Engineering Highlights

* **Fluid Masonry Layout:** Engineered using pure CSS column mechanics to create a responsive, Pinterest-style dynamic grid that adapts natively across varying desktop and mobile screen viewports.
* **Asynchronous Local Storage:** Implemented a debounced saving routine utilizing browser-native `localStorage` to preserve user configurations seamlessly across browser restarts.
* **Canvas Image Optimization:** Integrates a client-side performance feature that downscales and compresses high-resolution uploads via the Canvas API before rendering, protecting browser memory limits.
* **Decoupled Architecture:** Built completely from scratch with strict separation of concerns, separating semantic structure (`HTML5`), modular thematic layout design (`CSS variables`), and event-driven control logic (`Vanilla ES6+ JS`).

---

## 🛠️ Tech Stack & Tooling

* **Structure:** HTML5 (Semantic elements, Modal configurations, Contenteditable text fields)
* **Styling:** CSS3 (Variable styling tokens, Typography scale management, Radial background matrices)
* **Logic & Execution:** JavaScript ES6+ (Web APIs, File Reader transformations, Canvas scaling, UI State management)
